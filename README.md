# 🌍 Real-Time Air Quality Tracker

A Streamlit-based web application that provides real-time air quality and weather insights for any city using live APIs. The dashboard enables users to monitor pollutant levels, understand health risks, and view short-term forecasts through interactive visualizations.

---

## 🚀 Features

* 🔎 **Real-Time AQI Monitoring**

  * Displays AQI levels and key pollutants: PM2.5, PM10, CO, NO₂
  * Fetches live data using OpenWeatherMap and Open-Meteo APIs

* 🌤 **Weather Insights**

  * Temperature, humidity, and wind speed for the selected city

* 📊 **Pollutant Analysis**

  * Clean visual representation of pollutant levels
  * Status indicators (Good, Moderate, Unhealthy, etc.)

* 📈 **Forecasting & Prediction**

  * 48-hour pollutant forecast using Open-Meteo API
  * Short-term prediction using Linear Regression

* 🩺 **Health Recommendations**

  * Displays health impacts and safety suggestions based on pollutant levels

* 🪴 **Indoor Plant Suggestions**

  * Recommends air-purifying plants based on air quality

* 🤖 **AirBot Assistant**

  * Answers basic air quality-related queries

* 🎨 **UI Features**

  * Light/Dark theme toggle
  * Clean card-based layout for better readability

---

## 🛠 Tech Stack

* **Frontend & App Framework:** Streamlit
* **Backend & Data Processing:** Python
* **APIs:** OpenWeatherMap API, Open-Meteo API
* **Libraries:** Pandas, NumPy, Scikit-learn, Requests
